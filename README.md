# CodeNest

**CodeNest** is a student-focused learning platform inspired by GeeksforGeeks. It provides tutorials for programming languages, data structures, algorithms, web development, and databases. The website is built using HTML, CSS, JavaScript, and can be extended with Python/Flask for dynamic content management.

## Features

* GFG-style layout with a sidebar and main content area.
* Syntax-highlighted code blocks using Prism.js.
* Featured tutorials section.
* Categories sidebar for easy navigation.
* Smooth scrolling navigation.
* Fully responsive design.

## File Structure

```
CodeNest/
├── index.html      # Main homepage
├── style.css       # CSS styles
├── script.js       # JavaScript for dynamic behavior
├── README.md       # This file
└── tutorials/      # Optional: folder for Markdown tutorials (if using Flask)
```

## How to Use

1. **Open the website locally:**

   * Simply open `index.html` in your browser.

2. **Add Tutorials:**

   * For static site: Copy and add new tutorial cards in `index.html`.
   * For dynamic site: Use Flask with a `tutorials` folder to add Markdown files.

3. **Customize Styles:**

   * Modify `style.css` for colors, fonts, or layout adjustments.

4. **Enable Smooth Scrolling:**

   * `script.js` contains smooth scrolling and dynamic year display.

## Optional Python Integration

* Use Flask to serve tutorials dynamically.
* Tutorials can be written in Markdown and rendered automatically.
* Admin panel can be added for easy content management without touching HTML.
